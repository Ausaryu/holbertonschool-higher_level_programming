# python-input_output

## Description
File handling, JSON serialization, and object persistence.

## Projects
- `0-read_file.py`: Read and print a text file.
- `1-write_file.py`: Write text to a file.
- `2-append_write.py`: Append text to a file.
- `3-to_json_string.py`: Convert an object to JSON string.
- `4-from_json_string.py`: Parse JSON string to an object.
- `5-save_to_json_file.py`: Save an object to a JSON file.
- `6-load_from_json_file.py`: Load an object from a JSON file.
- `7-add_item.py`: Save list items from CLI to JSON.
- `8-my_class.py`: Define a simple class for serialization.
- `8-my_class_2.py`: Alternate class example for serialization.
- `8-class_to_json.py`: Convert class instance to JSON-ready dict.
- `9-student.py`: Create a serializable student class.
- `10-student.py`: Add selective attribute filtering.
- `11-student.py`: Add dynamic attribute updates.
- `12-pascal_triangle.py`: Generate Pascal's triangle.

## Why It Matters
These tasks teach persistence and structured data exchange in Python.
