# holbertonschool-higher_level_programming

## Description
This repository contains Python practice projects for higher-level programming concepts.

## Projects
- `pld exercises`: Miscellaneous practice exercises.
- `python-abc`: Abstract Base Classes and interface patterns.
- `python-classes`: Basics of classes and objects.
- `python-data_structures`: Lists and tuples.
- `python-exceptions`: Exception handling.
- `python-hello_world`: Python basics and first scripts.
- `python-if_else_loops_functions`: Control flow and functions.
- `python-import_modules`: Imports and command-line arguments.
- `python-inheritance`: Class inheritance.
- `python-input_output`: Files, JSON serialization, and persistence.
- `python-more_classes`: Advanced class features.
- `python-more_data_structures`: Sets and dictionaries.
- `python-test_driven_development`: Testing and doctest-driven design.
