# python-more_data_structures

## Description
Advanced data structures such as sets and dictionaries.

## Projects
- `0-square_matrix_simple.py`: Square matrix values.
- `1-search_replace.py`: Search and replace in a list.
- `2-uniq_add.py`: Sum unique values.
- `3-common_elements.py`: Find common elements in sets.
- `4-only_diff_elements.py`: Find symmetric differences.
- `5-number_keys.py`: Count keys in a dictionary.
- `6-print_sorted_dictionary.py`: Print a sorted dictionary.
- `7-update_dictionary.py`: Update dictionary entries.
- `8-simple_delete.py`: Delete a dictionary key.
- `9-multiply_by_2.py`: Multiply dictionary values.
- `10-best_score.py`: Find the best score in a dictionary.
- `11-multiply_list_map.py`: Multiply list values with map.
- `12-roman_to_int.py`: Convert Roman numerals to integers.

## Why It Matters
Efficient use of sets and dicts is key to clean, fast Python code.
