# python-inheritance

## Description
Exercises covering inheritance, method overriding, and subclassing.

## Projects
- `0-lookup.py`: Inspect available attributes and methods.
- `1-my_list.py`: Create a custom list class.
- `2-is_same_class.py`: Check if objects are exact instances.
- `3-is_kind_of_class.py`: Check instances using inheritance.
- `4-inherits_from.py`: Test subclass relationships.
- `5-base_geometry.py`: Define a base geometry class.
- `6-base_geometry.py`: Add area method and validation.
- `7-base_geometry.py`: Improve input validation.
- `8-rectangle.py`: Implement a rectangle class.
- `9-rectangle.py`: Enhance rectangle with string output.
- `10-square.py`: Implement a square class.
- `11-square.py`: Customize square string output.

## Why It Matters
Inheritance models real-world hierarchies and enables code reuse.
