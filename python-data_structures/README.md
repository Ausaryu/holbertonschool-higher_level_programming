# python-data_structures

## Description
Practice with core data structures such as lists and tuples.

## Projects
- `0-print_list_integer.py`: Print integers in a list.
- `1-element_at.py`: Retrieve an element safely by index.
- `2-replace_in_list.py`: Replace list values by index.
- `3-print_reversed_list_integer.py`: Print a list in reverse.
- `4-new_in_list.py`: Replace an element without mutating the original.
- `5-no_c.py`: Remove characters from a string.
- `6-print_matrix_integer.py`: Print a 2D matrix.
- `7-add_tuple.py`: Add two tuples element-wise.
- `8-multiple_returns.py`: Return length and first char of a string.
- `9-max_integer.py`: Find the max integer in a list.
- `10-divisible_by_2.py`: Map list values to divisibility booleans.
- `11-delete_at.py`: Delete an element by index.
- `12-switch.py`: Swap values in a list.

## Why It Matters
These tasks build fluency with list/tuple operations and defensive indexing.
