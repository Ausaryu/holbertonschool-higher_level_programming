# python-test_driven_development

## Description
Test-driven development with doctest and unit tests.

## Projects
- `0-add_integer.py`: Add integers with input validation.
- `2-matrix_divided.py`: Divide matrix values with checks.
- `3-say_my_name.py`: Format names with validation.
- `4-print_square.py`: Print a square with size checks.
- `5-text_indentation.py`: Format text with indentation rules.
- `6-max_integer.py`: Find max in a list.

## Why It Matters
TDD improves correctness, edge-case handling, and maintainability.
