# python-classes

## Description
Introductory exercises on classes, objects, attributes, and methods.

## Projects
- `0-square.py`: Define an empty class.
- `1-square.py`: Add size attribute with basic validation.
- `2-square.py`: Add area computation.
- `3-square.py`: Add property getters and setters.
- `4-square.py`: Add a `__str__`-like print behavior.
- `5-square.py`: Add comparison and size constraints.
- `6-square.py`: Add position handling for display.

## Why It Matters
Solidifies OOP fundamentals used throughout larger Python codebases.
