# python-import_modules

## Description
Using modules, packages, and command-line arguments.

## Projects
- `0-add.py`: Import and use a function.
- `1-calculation.py`: Use multiple imported functions.
- `2-args.py`: Print command-line arguments.
- `3-infinite_add.py`: Sum an arbitrary number of arguments.
- `4-hidden_discovery.py`: Discover names in a module.
- `5-variable_load.py`: Import a variable from a module.

## Why It Matters
Clean module usage and CLI handling are essential for reusable code.
