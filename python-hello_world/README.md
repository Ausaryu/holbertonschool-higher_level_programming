# python-hello_world

## Description
First steps in Python: basic syntax, scripts, and simple output.

## Projects
- `2-print.py`: Print a simple string.
- `3-print_number.py`: Format and print numbers.
- `4-print_float.py`: Format and print floats.
- `5-print_string.py`: Print a string multiple times.
- `6-concat.py`: Concatenate strings.
- `7-edges.py`: Slice strings by edges.
- `8-concat_edges.py`: Combine slices into a new string.
- `9-easter_egg.py`: Print the Python easter egg.

## Why It Matters
These tasks establish syntax comfort and the basic execution model of scripts.
