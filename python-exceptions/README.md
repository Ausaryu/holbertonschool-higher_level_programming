# python-exceptions

## Description
Exercises on handling errors with exceptions and `try/except` patterns.

## Projects
- `0-safe_print_list.py`: Print list items safely.
- `1-safe_print_integer.py`: Print integers with error handling.
- `2-safe_print_list_integers.py`: Print only integers from a list.
- `3-safe_print_division.py`: Safely divide values.
- `4-list_division.py`: Divide list elements with exception handling.
- `5-raise_exception.py`: Raise a type exception.
- `6-raise_exception_msg.py`: Raise an exception with a message.

## Why It Matters
Robust code depends on clear error handling and predictable failure modes.
