# python-if_else_loops_functions

## Description
Control flow, loops, and function basics in Python.

## Projects
- `0-positive_or_negative.py`: Classify numbers by sign.
- `1-last_digit.py`: Analyze the last digit of a number.
- `2-print_alphabet.py`: Print the alphabet.
- `3-print_alphabt.py`: Print alphabet with exclusions.
- `4-print_hexa.py`: Print numbers with hex values.
- `5-print_comb2.py`: Print combinations of two digits.
- `6-print_comb3.py`: Print combinations of three digits.
- `7-islower.py`: Check if a character is lowercase.
- `8-uppercase.py`: Convert a string to uppercase.
- `9-print_last_digit.py`: Print the last digit of a number.
- `10-add.py`: Add two integers.
- `11-pow.py`: Compute power of a number.
- `12-fizzbuzz.py`: Implement FizzBuzz.

## Why It Matters
These exercises build core control flow skills used in everyday programming.
